# 패키지 만들기



### 패기지란 무엇인가?
##### 패지니는 도트를 이용하여 파이썬 모듈을 계층적(디렉터리 구조)로 관리 할 수 있게 해준다.


###### 패키지 예
```
Libs/
    common/
        packet.py
    database/
        query.py
    test/

main.py
```

###### 주의
```
※ python3.3 버전부터는 __init__.py 파일 없이도 패키지로 인식이 된다. 
   하지만 하위 버전 호환을 위해 __init__.py 파일을 생성하는 것이 안전한 방법이다.
```


###### 해당 내용은 점프 투 파이썬에서 발췌 했음
